## 🎯 初回開発タスクの進め方

### 📝 **計画フェーズ**

1. **要求理解**: 何を作るかを明確化
2. **影響範囲**: どのファイルを変更するか
3. **品質確認**: テスト・ドキュメント更新の要否

### ⚒️ **実装フェーズ**

1. **型定義更新**: OpenAPI仕様更新（必要に応じて）
2. **コード実装**: フロントエンド・バックエンド
3. **品質チェック**: `pnpm type-check && pnpm lint && pnpm build`

### 🎉 **完了フェーズ**

1. **動作確認**: 実際にブラウザで動作確認
2. **PR作成**: [貢献ガイドライン](../contrib/contribution-guide.md)参照
3. **レビュー対応**: フィードバック反映
